# Hellwave Game Data

## Maps

### Doom

The second map to experiment with WAD3 texture support as well as PBR materials. It’s designed after Classic Doom’s E1M7.

### E1M2

The first map to validate the game concept. It’s based on Quake’s E1M2, some improved lighting and slight changes to incorporate the new game mechanics as well as some creative doodling.
